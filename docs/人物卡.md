# 游戏初始人物卡设计方案 (Character Cards Blueprint)

本方案包含 8 张极具差异化、属性配置激进的初始人物卡。设计核心在于**拉大属性两极分化（上限 100）**，以确立鲜明的生态位。所有人物卡**开局均不自带任何初始工具**，完全依赖极端的属性或初始资源来驱动不同的局内破局路线。

---

## 1. 象牙塔学霸 (Academy Prodigy)

> **角色背景**：理论与算法的巨子，学习效率拉满。但温室里的花朵从未面对过真实的生产环境，抗压和沟通能力极度匮乏，极易在连环 Bug 中破防。

### 📊 属性配置

| 属性 id | 名称 | 属性值 |
| --- | --- | --- |
| **learning** | 学习 | **86** |
| **logic** | 逻辑 | **62** |
| `focus` | 专注 | 40 |
| `creativity` | 创造 | 30 |
| `resilience` | 抗压 | **34** |
| `communication` | 沟通 | **15** |

### 🎒 初始配置

* **正向资源**：`knowledge` +30（开局自带大量理论储备）
* **负面限制**：`money` -20（贫穷的在校生，经济窘迫）
* **初始技能**：无（纯靠 86 的恐怖学习率在开局自行快速领悟）
* **活动等级**：`study` (系统学习) 升至 **Lv.2**
* **初始工具**：无
