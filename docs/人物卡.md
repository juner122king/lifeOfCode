# 游戏初始人物卡设计方案 (Character Cards Blueprint)

本方案包含 8 张极具差异化、属性配置激进的初始人物卡。设计核心在于**保留鲜明生态位，但收紧活动等级奖励**：大多数人物卡只靠属性、资源和技能区分，活动等级只给最能代表开局路线的少数角色。

## 字段约定

* **attributes**：写入 `state.attributes` 的实际初始基础属性，不是 100 分风格评分。高优势属性约 `64-72`，次优势约 `44-60`，普通约 `18-36`，弱项约 `2-16`。
* **resources**：基于默认开局资源的增减 delta。负资源必须应用后仍不低于 0，不使用会被钳制失效的惩罚。
* **skills**：初始技能等级，至少 Lv.1。
* **activityLevels**：只记录额外提升；未列出即全部默认 Lv.1。
* **ownedTools**：本轮统一为空，所有人物卡开局均不自带工具。

## 活动等级原则

* 活动等级奖励遵循“宁少勿滥”。
* 只保留 3 张卡拥有活动 Lv.2：`study` 只给学霸，`feature-coding` 只给独立开发者，`rest` 只给面条哥。
* 其余 5 张卡不额外提升活动等级。
* 不做三级活动，也不赠送需要多段前置的活动。

## 1. 象牙塔学霸 (Academy Prodigy)

> **角色背景**：理论与算法的巨子，学习效率拉满。但温室里的花朵从未面对过真实的生产环境，抗压和沟通能力极度匮乏，极易在连环 Bug 中破防。

### 📊 属性配置

| 属性 id | 名称 | 属性值 |
| --- | --- | --- |
| **learning** | 学习 | **72** |
| **logic** | 逻辑 | **58** |
| `focus` | 专注 | 32 |
| `creativity` | 创造 | 20 |
| `resilience` | 抗压 | **10** |
| `communication` | 沟通 | **4** |

### 🎒 初始配置

* **正向资源**：`knowledge` +80（开局自带大量理论储备）
* **负面限制**：`money` -10（贫穷的在校生，经济窘迫）
* **初始技能**：无（纯靠机制内极高学习属性在开局自行快速领悟）
* **活动等级**：`study` (系统学习) 升至 **Lv.2**
* **初始工具**：无
